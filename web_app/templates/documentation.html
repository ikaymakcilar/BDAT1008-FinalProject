Welcome to the API Documentation

This documentation provides information on how to use the API endpoints.

API Endpoints

Get All Predictions and Real Closing Values:

Retrieves a list of all items.

            http://127.0.0.1:8000/api/predictions

Get Prediction by Date:

GET /api/prediction/[date];

Retrieves the prediction data by date.

Example Request:

            http://127.0.0.1:8000/api/prediction/2024-01-24

Example Output:

            {
                "predicted_close": "121.417",
                "real_close": "120.630"
            }
